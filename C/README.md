# C Program

This is a basic C program skeleton.

## Building the Program

To build the program using make:

```bash
make
```

## Running the Program

To run the program:

```bash
./program
```

Or build and run in one command:

```bash
make run
```

## Cleaning Build Artifacts

To clean up compiled files:

```bash
make clean
```

## Requirements

- GCC compiler
- Make
